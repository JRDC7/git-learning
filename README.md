# Git Learning Repository

This repository documents my **step-by-step learning journey with Git and GitHub**.  
It includes all core concepts, workflows, and practical exercises completed through official tutorials.

---

## üìù Contents

1. **Installation & Configuration**
   - Installing Git
   - Setting username & email
   - Setting default branch (`main`)

2. **Repository Setup**
   - Initializing a repository (`git init`)
   - Creating and tracking a README file
   - Staging changes & making commits

3. **Commit Workflow**
   - Recognizing changes in the working directory
   - Staging files (`git add`)
   - Creating commits (`git commit -m "message"`)
   - Viewing commit history (`git log --oneline --graph`)

4. **Undo & Restore**
   - Unstaging changes (`git restore --staged <file>`)
   - Discarding changes (`git restore <file>`)
   - Resetting commits (`git reset --soft HEAD~1`)

5. **Branching & Merging**
   - Creating and switching feature branches (`git branch` / `git checkout`)
   - Committing changes in branches
   - Merging branches (`git merge`)
   - Detecting and resolving conflicts

6. **GitHub Collaboration**
   - Pushing branches to GitHub (`git push -u origin <branch>`)
   - Creating & merging Pull Requests
   - Synchronizing local repository (`git pull origin main`)
